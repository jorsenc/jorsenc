¿Por qué necesitas esto?

Cada vez que abres una sesión nueva con Claude Code, empieza desde cero. No recuerda lo que hicisteis ayer, ni tu negocio, ni tus proyectos. Tienes que explicárselo todo de nuevo.

NotebookLM es la herramienta gratuita de Google que actúa como su memoria. Claude Code le pregunta lo que necesita y NotebookLM le responde al instante, sin gastar tokens. Tú no pagas nada extra.

1
Instalar la conexión

Copia este prompt, ábrelo en Claude Code y pégalo. Él se encargará de instalar todo automáticamente. Solo tendrás que hacer una cosa: entrar con tu cuenta de Google cuando te lo pida.

📋 Prompt de instalación — copia y pega en Claude Code
Necesito que instales y configures el servidor MCP de NotebookLM para que puedas usarlo desde aquí. Usa el paquete notebooklm-mcp-cli del repositorio de jacob-bd en GitHub. Encárgate tú de todo el proceso: instalar el paquete, configurar el servidor MCP en mis archivos de configuración y preparar el skill experto para que sepas cómo usar NotebookLM. Cuando llegues al paso de autenticación con Google, abre el navegador y dime exactamente qué tengo que hacer para iniciar sesión. Guíame paso a paso en ese momento. Al terminar, verifica que la conexión funciona listando mis notebooks de NotebookLM y dime cuántos tienes.
💡 Claude Code puede tardar 2-3 minutos instalando cosas en segundo plano. Es normal. No cierres la ventana.
⚠️ En el momento de autenticación, Claude Code te dirá que abras el navegador. Hazlo, entra con tu cuenta de Google de siempre y cierra el navegador cuando veas tus notebooks. Eso es todo.
✅ Sabrás que funciona cuando Claude Code te liste el nombre de tus notebooks de NotebookLM.
2
Crear tu Cerebro Maestro

Este es el notebook permanente donde Claude Code guardará todo lo que sabe sobre ti. Se crea una sola vez y va creciendo con el tiempo. Sustituye los datos en mayúsculas por los tuyos.

📋 Prompt — Cerebro Maestro
Usando NotebookLM, comprueba si ya existe un notebook llamado "Cerebro Álvaro". Si no existe, créalo. Después añade como fuente de texto este contexto sobre mí: - Nombre: Álvaro Morales - Empresa: Automatizado Agency, agencia de inteligencia artificial - Canal de YouTube: @AlvaroMoralesIA - Herramientas que uso: Make, n8n, Claude, Supabase, Notion - Mi audiencia: emprendedores y profesionales hispanohablantes de 35 a 54 años, sin conocimientos técnicos, que quieren implementar IA en sus negocios - Mi historia: pasé de trabajar en una fábrica y como vigilante nocturno a CEO de una agencia de IA con clientes internacionales - Objetivo de mi canal: enseñar a usar la IA en negocios reales sin necesidad de saber programar Cuando termines, dime que el notebook está creado y confírmame su nombre exacto.
🧠 A partir de ahora, añade al inicio de tus proyectos en Claude Code esta instrucción: "Antes de responder sobre mi negocio o estrategia, consulta siempre el notebook 'Cerebro Álvaro' en NotebookLM."
3
La demo del vídeo — Investigación + Infografía

Este es el prompt que usas en directo en el vídeo. Muestra el flujo completo: busca vídeos en YouTube, los analiza en NotebookLM y genera una infografía. Todo con un solo mensaje.

📋 Prompt demo — el que grabas en el vídeo
Quiero que hagas lo siguiente en orden y me confirmes cada paso antes de continuar: PASO 1 → Busca en YouTube los 10 mejores vídeos sobre "automatización de procesos para pequeñas empresas". Muéstrame el listado con título, canal y enlace. PASO 2 → Crea un notebook nuevo en NotebookLM llamado "Investigación - Automatización PYMES" y añade esos 10 vídeos como fuentes. Avísame cuando estén todos cargados. PASO 3 → Pídele a NotebookLM que analice esos vídeos y te diga: los 5 procesos que más se mencionan para automatizar, qué herramientas aparecen más y qué duda tiene la gente que nadie responde bien. PASO 4 → Genera una infografía en NotebookLM con ese análisis. Fondo oscuro, tipografía clara, estilo moderno. Guárdala en mi carpeta de Descargas y dime exactamente dónde está el archivo.
🔍
Claude Code busca en YouTube
Sin que tú toques nada. Saca títulos, canales y URLs de los mejores vídeos del tema.
📓
Crea el notebook y carga las fuentes
Manda automáticamente todos los vídeos a NotebookLM. Tú no copias ni pegas nada.
🧩
NotebookLM analiza el contenido
Google hace el trabajo pesado gratis. Claude Code no gasta tokens en esto.
🖼
Aparece la infografía en tu ordenador
El archivo se guarda directamente en tu carpeta de Descargas. Lista para mostrar.
4
Guardar memoria al acabar cada sesión

Al terminar cualquier sesión importante de trabajo, pega este prompt. Claude Code resumirá todo lo que hicisteis y lo guardará en tu Cerebro Maestro. La próxima vez lo recordará todo sin que tengas que explicar nada.

📋 Prompt de wrap-up — al final de cada sesión
Haz un resumen de todo lo que hemos trabajado en esta sesión. Incluye: - Qué hemos hecho - Decisiones importantes que hemos tomado - Tareas que quedan pendientes - Cualquier cosa que deba recordar la próxima vez Guarda ese resumen como nueva fuente de texto en el notebook "Cerebro Álvaro" de NotebookLM. El título del documento debe ser la fecha de hoy. Cuando lo hayas guardado, dime el nombre exacto del documento.
✅ Con esto, cada sesión que tengas con Claude Code queda guardada en NotebookLM. La próxima vez que abras Claude Code, consultará ese historial antes de responderte. Nunca más empezará desde cero.
🔄 Una vez cada 2-4 semanas, Claude Code te avisará de que la sesión con NotebookLM ha expirado. Si eso pasa, simplemente dile: "Renueva la conexión con NotebookLM" y él abrirá el navegador para que entres con Google de nuevo.

Ya tienes todo lo que necesitas 🎉
